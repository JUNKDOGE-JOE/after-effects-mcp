# ae-mcp-snapshot-mss

## 中文

`ae-mcp-snapshot-mss` 是 ae-mcp 的跨平台截图实现，基于 [mss](https://python-mss.readthedocs.io/) 和 Pillow。

它支持 Windows、macOS 和 Linux。当前实现按屏幕区域截图；Windows 下可通过 HWND 转换到窗口矩形。

### 安装

```powershell
pip install ae-mcp-snapshot-mss
```

### 协议

MIT。

## English

`ae-mcp-snapshot-mss` is the cross-platform screenshot implementation for ae-mcp, built on [mss](https://python-mss.readthedocs.io/) and Pillow.

It supports Windows, macOS, and Linux. The current implementation captures by screen rectangle; on Windows, HWND can be translated to a window rectangle.

### Install

```powershell
pip install ae-mcp-snapshot-mss
```
